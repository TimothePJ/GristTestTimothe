# Synchro

Page de synchronisation entre :

- `Planning Projet`
- `gestion-depenses2`

La page embarque les deux plannings dans une meme vue, avec une frise commune en haut. A la selection d'un projet, la vue se cale sur un an a partir du premier segment de la premiere ligne rendue dans `Planning Projet`, puis applique le meme viewport au plan de charge de `gestion-depenses2`.

Le planning `Planning Projet` reste en consultation seule en mode embarque. Le planning `gestion-depenses2` conserve son bouton `Editer` et son fonctionnement existant.
