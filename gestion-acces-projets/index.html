Configuration Grist requise

Le widget administre la colonne Team.Acces_Lecture_Projets. Les règles d’accès Grist restent responsables de la sécurité réelle.

  1. Créer la colonne Acces_Lecture_Projets dans Team, de type Choice List.
  2. Créer une formule Cle_Acces dans chaque table protégée contenant le projet et le service.
  3. Autoriser R lorsque la clé de la ligne appartient à user.TeamRec.Acces_Lecture_Projets.
  4. Réserver la modification de la colonne de droits aux propriétaires et aux lignes Team.Admin.

Les formules exactes et l’ordre des règles sont détaillés dans le fichier README du widget.
